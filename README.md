# Cookie-Clicker


Pour lancer le Cookie Clicker de l'équipe 2, vous pouvez l'accéder à cette adresse : 
https://cookie-clicker-equipe-2.onrender.com/



Pour lancer le code en local faut avoir installé Node.js;
    avoir installé express avec `npm install express`;
    Lancer la commande `node server.js` puis suivre es instruction sur le terminal



Bug observés (en guise d'avertissement à l'utilisateur) :
-Possibilité de overflow si on a un trop grand nombre de cookies
-Le leaderboard s'affiche mal avec des navigateur comme brave
